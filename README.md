# ESP32 Weather Monitor (ST7735 1.8inch)

ESP32 と ST7735S 1.8インチ TFT 液晶ディスプレイを使用した気象情報ステーションです。  
日本全国 47 都道府県庁所在地の天気を表示し、NHK ニュース速報も確認できます。

---

## 1. システム構成

### 1.1 使用ライブラリ

| 機能 | ライブラリ |
| :--- | :--- |
| TFT 描画 | Adafruit_GFX / Adafruit_ST7735 |
| 日本語フォント | U8g2_for_Adafruit_GFX |
| HTTP 通信 | HTTPClient |
| JSON パース | ArduinoJson |
| 時刻管理 | time.h（NTP・JST） |
| WiFi 設定 | WiFiManager（キャプティブポータル） |
| 設定保存 | Preferences（ESP32 NVS） |
| Web 設定 UI | WebServer（ESP32 組み込み） |
| RSS 解析 | 自前 XML パーサ |

### 1.2 ファイル構成

```
src/
  config.h        ピン/レイアウト/色/タイマー定数
  cities.h/cpp    47都道府県庁所在地データ（緯度・経度・地方ブロック）
  weather.h/cpp   WMO変換・背景色・アイコン・週間&毎時描画
  display.h/cpp   天気画面描画（差分更新対応）
  network.h/cpp   WiFi・NTP・Open-Meteo API
  ticker.h/cpp    ニュース画面（描画 + Core 0 バックグラウンドタスク）
  button.h/cpp    3ボタン制御（短押しのみ）
  settings.h/cpp  Web 設定ページ・NVS 保存・地方フィルタ
  main.cpp        setup() / loop()
```

## 2. ハードウェア配線

### ディスプレイ（ST7735S）

| TFT 端子 | ESP32 GPIO | 備考 |
| :--- | :--- | :--- |
| VCC | 3.3V | 電源 |
| GND | GND | グランド |
| CS | 5 | Chip Select |
| RESET | 4 | Hardware Reset |
| D/C (A0) | 2 | Data/Command |
| SDA | 23 | SPI MOSI |
| SCLK | 18 | SPI Clock |
| BL | 3.3V | バックライト常時点灯 |

### ボタン（3個・全て短押しのみ）

| ボタン名 | GPIO | 配線 | 備考 |
| :--- | :--- | :--- | :--- |
| SCREEN | 0 | GPIO 0 ↔ GND | 内部プルアップ使用 |
| MODE | 35 | GPIO 35 ↔ GND | 外付け 10kΩ プルアップ推奨 |
| NEXT | 34 | GPIO 34 ↔ GND | 外付け 10kΩ プルアップ必要 |

> **注意**: GPIO 34・35 は入力専用ピンのため内部プルアップが効きません。  
> 10kΩ 抵抗で 3.3V に接続する外付けプルアップが必要です。

## 4. ボタン操作

長押しを廃止し、3 ボタン・短押しのみのシンプルな操作体系です。

| ボタン | WEATHER 画面（地方巡回） | WEATHER 画面（自分の都市） | NEWS 画面 |
| :--- | :--- | :--- | :--- |
| **SCREEN** | ニュース画面へ | ニュース画面へ | 天気画面へ |
| **MODE** | → 自分の都市モードへ | → 地方巡回モードへ | 無効 |
| **NEXT** | 次の都市（地方フィルタ順） | 週間 ⇄ 毎時 切替 | 次の見出し |

---

## 5. 表示モード

```text
WEATHER 画面
 ├── 地方巡回モード ── 選択した地方の都市を 20 秒ごとに自動切替
 │                     NEXT ボタンで手動スキップ可
 └── 自分の都市モード ── Web 設定で登録した都市を固定表示
      ├── 週間予報 ── 7 日間（1 時間ごと更新）
      └── 毎時予報 ── 6 時間分（30 分ごと更新）

NEWS 画面 ── NHK RSS から最大 20 件の見出しを全画面表示（5 秒自動切替）
```

## 6. Web 設定ページ

WiFi 接続後、ブラウザから `http://<ESP32のIPアドレス>/` にアクセスすると設定画面が開きます。

### 設定項目

| 項目 | 内容 |
| :--- | :--- |
| **自分の都市** | 自分の都市モードで表示する都市（47 都市から選択） |
| **巡回する地方** | 地方巡回モードで巡回するエリア |

### 地方フィルタの選択肢

| 設定値 | 対象都市 |
| :--- | :--- |
| 全国 | 全 47 都市 |
| 北海道・東北地方 | 札幌・青森・盛岡・仙台・秋田・山形・福島 |
| 関東地方 | 水戸・宇都宮・前橋・さいたま・千葉・東京・横浜 |
| 中部地方 | 新潟・富山・金沢・福井・甲府・長野・岐阜・静岡・名古屋 |
| 関西地方 | 津・大津・京都・大阪・神戸・奈良・和歌山 |
| 中国・四国地方 | 鳥取・松江・岡山・広島・山口・徳島・高松・松山・高知 |
| 九州・沖縄地方 | 福岡・佐賀・長崎・熊本・大分・宮崎・鹿児島・那覇 |

設定は「保存する」ボタンで NVS（フラッシュ）に書き込まれ、次回起動後も保持されます。  
保存直後に天気データが取得し直され、ディスプレイへ即時反映されます。

## 7. データ更新タイミング

| データ | 更新間隔 |
| :--- | :--- |
| 現在天気（気温・天気コード） | 15 分 |
| 週間天気予報 | 1 時間 |
| 毎時天気予報 | 30 分 |
| 都市自動切替（地方巡回） | 20 秒 |
| 時計表示 | 30 秒 |
| NHK ニュース RSS | 5 分（バックグラウンド） |
| ニュース自動ページ送り | 5 秒 |

## 8. 起動フロー

```text
ディスプレイ初期化
 → WiFiManager（初回は AP モードでスマホから設定）
 → NTP 時刻同期（JST）
 → NVS から設定読み込み（自分の都市・地方フィルタ）
 → Web 設定サーバー起動
 → 地方フィルタで最初の都市を決定
 → 初回天気データ取得 → 画面表示
 → NHK ニュース取得タスク起動（Core 0）
 → ループ開始
```
